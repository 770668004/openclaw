# 技能管理器 (Skill Manager)

这个技能提供了一个简单的方式来查询和管理所有已安装的OpenClaw技能。

## 功能特点

- **分类展示**：将58个技能分为11个逻辑类别，便于理解和查找
- **简洁描述**：每个技能都有通俗易懂的功能说明
- **快速搜索**：支持关键词搜索特定技能
- **详细信息**：可以查看特定分类或单个技能的详细信息

## 使用方法

### 1. 查看所有技能分类概览
```bash
skill-manager
# 或
list-skills
```

### 2. 查看特定分类的技能详情
```bash
skill-manager "系统安全与审计"
skill-manager "开发与代码管理"
```

### 3. 搜索特定技能
```bash
skill-manager --search github
skill-manager --search weather
```

### 4. 查看所有技能完整列表
```bash
skill-manager --all
```

## 分类说明

- **系统安全与审计**：系统安全加固、备份恢复等
- **开发与代码管理**：GitHub集成、代码生成、性能优化等  
- **多媒体与娱乐**：音乐控制、视频处理、AI图像生成等
- **通信与消息**：各种消息平台集成
- **生产力工具**：笔记、任务管理、密码管理等
- **智能助手功能**：天气、搜索、摘要、预测等
- **硬件与设备控制**：智能家居、摄像头等设备控制
- **AI与模型工具**：语音识别、TTS、模型统计等
- **文件与数据处理**：文件维护、PDF处理、规划等
- **技能管理与发现**：技能创建、发现新技能等
- **系统工具**：会话管理、日志管理、邮件客户端等

## 安装

这个技能已经自动安装在您的workspace中，无需额外操作。

## 更新

要更新技能列表，请重新运行安装脚本：
```bash
~/.openclaw/workspace/skills/skill-manager/scripts/install.sh
```
